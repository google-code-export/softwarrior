2          Short program description

The program - bit torrent client (like uTorrent), with additional search capabilities, and downloading content from the site rutracker.org and from others sites.

The goal of the program - to make handy and faster torrent client for Android.

Note: If you like program and you would like to use it without ads, please buy in Android Market Rutracker Downloader Donate. It is patch to remove Ads from Rutracker Downloader.  Thanks in advance.

Functions of the program:

·        «Pirate Search»-  to search and download torrent files from sites:  piratebay.org, www.mininova.org, isohunt.com

·        Download of the content (inside torrent client)

·        «Torrent Manager» - to manage multiple downloading

+ Select files - you can choose which files to download, for example, if the torrent file is generated for several albums of music.

·        Program processes all torrent files (to push torrent file in file manager and program will start download)

·        Manage of the space storage of disc (inside file manager)

·        WEB search of the content on sites.

·        RSS search of the content on site (only rutracker.org)

·        View of the site map

The inside torrent client is started as a service and it does not demand program to be constantly active while loading.

Main Menu of the program:

·        About program – window with info about program and contact information.

·        Help – user manual

·        Preferences – search preferences, torrent, proxy, UPnP service, Local Service Discovery, NAT-PMP service

·        File Manager – inside file manager. Allows to mage downloaded content right in the program.  

·        Exit – exit from the program.

4         User manual

4.1      Common description

The program doesn’t demand additional preference and ready to work right after installation.  To choose torrent safe path is recommended. it’ll be program work catalogue. Root of SD Card is set up by default.  

Make sure that your device has Internet connection at the moment and there is space on SD Card.

To start a search on rutracker.org you need to log in.

Press the key “Log in”, log user name and password in and press “Enter”. In case of success you skip to the main tab of the website.

            Press the key “back to search”. Name search options. Press the key “Search”.

In case you chose “WEB search” you skip to inside “WEB browser”

In case you chose ”WEB search” – “Show Site Map” you skip to inside “WEB Browser” that opens you the map of the site.

4.2      Download

The screen is managed by torrent client. In the screen the following is possible:

-          «Torrent Manager» - to manage multiple downloading

-          To start downloading (start)

-          To stop downloading (stop)

-          To pause downloading (pause)

-          To resume downloading (resume)

-          To select files (Files) - you can choose which files to download, for example, if the torrent file is generated for several albums of music.

-          To push torrent file in file manager and program will start download

-          Option "Allocate storage" - allows you to allocate space for the file before downloading. For larger files, allocation requires much time and resources. The enabled option speeds up the downloading because to do minimum disk operations. Only with the option enabled, you can choose files to download. For large files

(> 750 MB) NOT RECOMENDED to enable option.

In the screen you can monitor downloading process in details.

Torrent Client is made as a service so you can easily send program to background, if press a phone key «Home»

To back and to check the downloading process you can at any moment – press a key “icon” on status panel of the phone.

When the content is loaded Torrent Client is automatically switches to seeding mode.

NB. Functional abilities of the Client that are available in the program are not the only options. The Client may work as a true torrent client (uTorrent). Interface can be extended by users’ requests.

4.3      WEB search

Installed into program WEB browser is adopted to work with rutracker.org, pornolab.net and nnm-club.ru

After WEB search site with search results is shown in the screen.

You can use the site as in usual Browser.

If you go to the tab with ready distribution, WEB Browser offers to load distribution.

If you press the key “Download torrent” you skip to inside “Torrent Client”.

4.4      RSS search

On the screen RSS is shown with contents that you’ve chosen in the search.

While you press RSS client you skip to inside WEB browser with ready distribution there you can download torrent file.

If you press phone key “Back” you back to RSS Client. 

5         Usage recommendations

·        If the search does not work - make sure you are logged on the. Repeat the procedure for login.

·        Look closely at how much free space on your SD card and a volume of distribution that you are downloading.

·        DO NOT FORGET, SD cards are generally formatted in FAT32. And he has a limitation on the maximum file size 4 gigabytes, EXT2(16Gb).

·        If you do not put limits on downloads Torrent client will try to get the maximum download speed from WIFI card. If speed will be more than 3 Megabits per second built-in WIFI card devices are beginning to heat up. The higher the speed the higher power consumption. The battery will also be quickly discharged. If you bet on downloading a large file then connect your phone to the charger and put it on the open surface.
